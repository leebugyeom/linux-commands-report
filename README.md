# 리눅스 명령어 조사: top, jobs, kill

## 1. 명령어 요약

| 명령어 | 기능 요약 |
|--------|-----------|
| top    | 시스템의 실시간 프로세스 및 자원 사용량 모니터링 |
| jobs   | 현재 셸에서 실행 중인 백그라운드 작업 조회 |
| kill   | 프로세스 종료 또는 신호 전송 |

---

## 2. top 명령어

### 개요
top 명령어는 CPU, 메모리, 프로세스 정보를 실시간으로 확인하는 시스템 모니터링 도구이다.

### 주요 옵션

| 옵션 | 설명 |
|------|------|
| -d   | 업데이트 주기 설정 |
| -p   | 특정 PID만 표시 |
| -u   | 특정 사용자 프로세스만 표시 |

## 3. jobs 명령어

### 개요
jobs 명령어는 현재 셸에서 실행 중이거나 중지된 백그라운드 작업 목록을 출력한다.

### 사용 예시

```bash
$ jobs
[1]   Running    python server.py &
[2]   Stopped    vim

## 4. kill 명령어

### 개요
kill 명령어는 프로세스에 종료 신호(signal)를 보내는 명령어이다.

### 대표적인 시그널
| 시그널 | 번호 | 설명 |
|--------|------|------|
| SIGTERM | 15 | 정상 종료 요청 |
| SIGKILL | 9  | 강제 종료 |
| SIGHUP  | 1  | 설정 재시작 |

### 사용 예시
```bash
kill -9 1234
